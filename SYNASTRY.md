# 合盘功能说明

合盘分析（Synastry）是占星学中用来分析两个人之间关系的重要工具。通过比较两个人的星盘，可以揭示出彼此之间的动态关系、互动模式和潜在的挑战。

## 功能特点

1. **合盘计算**：基于两个人的星盘数据，计算行星之间的相位关系
2. **兼容性评分**：从多个维度评估两人的兼容性
3. **关系洞察**：提供关系互动方式的深入解读
4. **相位分析**：识别最重要的行星相位，及其对关系的影响

## 使用方法

1. 在星盘页面点击"合盘分析"按钮
2. 选择两个不同的星盘档案
3. 点击"计算合盘"按钮
4. 查看合盘分析结果

## 技术实现

合盘功能由以下部分组成：

1. **后端API**：`/api/synastry/calculate`
   - 接收两个档案ID
   - 调用`calculate_synastry`计算合盘结果
   - 返回合盘数据和解读

2. **前端界面**：
   - `synastry.html`：档案选择页面
   - 合盘结果展示：显示兼容性分数、相位列表和解读文字

3. **合盘计算**：`calculate_synastry`函数（在`astro_utils.py`中）
   - 分析两个星盘中行星之间的相位
   - 计算各个维度的兼容性分数
   - 生成关系解读文字

## 兼容性维度

合盘分析提供以下几个维度的兼容性评分：

1. **整体兼容**：两人星盘的综合兼容性
2. **爱情**：情感和浪漫关系的潜力
3. **沟通**：思想交流的顺畅程度
4. **价值观**：核心价值观的相似性
5. **情感**：情感表达的匹配程度
6. **友谊**：友谊基础的稳固程度

## 相位类型

在合盘分析中，行星之间的相位关系被分为以下几类：

1. **合相（0°）**：能量融合，强烈影响
2. **六分相（60°）**：和谐合作，积极流动
3. **四分相（90°）**：紧张对立，成长挑战
4. **三分相（120°）**：自然和谐，轻松流动
5. **对分相（180°）**：两极拉扯，平衡挑战

相位的影响强度取决于相位类型、所涉及的行星，以及相位的精确度。
