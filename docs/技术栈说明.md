# 技术栈说明

## 后端框架

| 技术 | 作用 |
|------|------|
| **FastAPI** | Web 框架，提供所有 REST API 接口和 WebSocket 支持 |
| **Uvicorn** | ASGI 服务器，运行 FastAPI 应用 |
| **SQLModel** | ORM 框架，定义数据库模型并操作 MySQL |
| **PyMySQL** | MySQL 数据库驱动 |
| **sqladmin** | 基于 FastAPI 的后台管理面板，用于查看和管理数据库数据 |

## 认证与安全

| 技术 | 作用 |
|------|------|
| **python-jose** | 生成和验证 JWT Token，实现用户登录态 |
| **passlib / bcrypt** | 密码哈希加密存储 |
| **cryptography** | 底层加密支持 |

## AI 与 LLM

| 技术 | 作用 |
|------|------|
| **LangChain** | AI 应用框架，串联 LLM、工具调用、对话记忆 |
| **langchain-openai** | 对接 SiliconFlow（OpenAI 兼容接口），用于 LLM 对话和向量 Embedding |
| **SiliconFlow API** | 提供 LLM 推理（Qwen/DeepSeek 等模型）和 BAAI/bge-m3 向量化 |

## RAG 知识库

| 技术 | 作用 |
|------|------|
| **LanceDB** | 本地向量数据库，存储知识库文档的向量索引 |
| **langchain-community** | 提供 LanceDB 向量库的 LangChain 封装 |
| **pypdf** | 解析上传的 PDF 文档 |
| **python-docx** | 解析上传的 Word 文档 |

## 星盘计算

| 技术 | 作用 |
|------|------|
| **flatlib** | 西洋占星学库，计算星盘行星位置、宫位、相位 |
| **pytz** | 时区转换，将用户本地时间转为 UTC 用于精确星盘计算 |

## 实时通信

| 技术 | 作用 |
|------|------|
| **FastAPI WebSocket** | 实现用户之间的实时聊天功能 |

## 前端

| 技术 | 作用 |
|------|------|
| **React（ReAct）** | AI Agent 推理机制，LangChain 用其实现"思考→工具调用→观察→再思考"的循环，驱动占星师 AI 完成多步推理和工具使用 |

## 工具库

| 技术 | 作用 |
|------|------|
| **httpx** | 异步 HTTP 客户端，调用外部 AI API |
| **python-dotenv** | 从 `.env` 文件加载环境变量（API Key 等配置） |
| **Pillow** | 图片处理（头像上传等） |
| **aiofiles** | 异步文件读写 |
| **python-multipart** | 处理文件上传的 multipart 表单 |
